# en.dev

en.dev builds open source developer tools for making everyday development smoother, faster, and more reliable.

Our main project is [mise](https://github.com/jdx/mise), a development environment manager for languages, CLIs, tasks, and project workflows. mise helps developers and teams keep their tools consistent across projects.

## Support

en.dev is supported by members and sponsors. That support makes it possible to keep maintaining mise, shipping releases, improving docs, supporting the community, and building new tools in the open.
